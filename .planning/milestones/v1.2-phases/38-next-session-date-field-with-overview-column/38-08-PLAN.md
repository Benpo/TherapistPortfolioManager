# The UAT Gaps section hints "the simplest mirror is likely dropping the early
# returns and letting aNext.localeCompare(bNext) handle blanks like lastSession".
# That hint is WRONG for this column and must NOT be followed literally. Because

# requirement is "bottom on ascending, top on descending" (see the UAT gap truth
# and the decision-revision note). Implement THAT via the far-future-sentinel
# approach in Task 1 — do not just delete the early returns.
