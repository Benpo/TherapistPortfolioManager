**CRITICAL for globe switcher on disclaimer pages:** When switching language, the `?readonly` and `?next` params must be preserved. The onLangChange callback must read `window.location.search` and append it to the new URL:
```javascript
onLangChange: function(newLang) {
  try { localStorage.setItem('portfolioLang', newLang); } catch(e) {}
  var params = window.location.search || '';
  var base = newLang === 'de' ? './disclaimer.html' : './disclaimer-' + newLang + '.html';
  window.location.href = base + params;
}
```
