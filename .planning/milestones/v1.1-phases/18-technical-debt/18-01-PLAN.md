**In assets/license.js:**

1. Add two helper functions near the top (after the LICENSE_I18N object, before getLicenseLang):

```javascript
function encodeLicenseValue(val) {
  try { return btoa(val); } catch (e) { return val; }
}

function decodeLicenseValue(encoded) {
  try { return atob(encoded); } catch (e) { return encoded; }
}
```

2. In `activateLicenseKey()` (around line 181), change the three localStorage.setItem calls:
   - `localStorage.setItem('portfolioLicenseKey', encodeLicenseValue(key));`
   - `localStorage.setItem('portfolioLicenseInstance', encodeLicenseValue(data.instance.id));`
   - `localStorage.setItem('portfolioLicenseActivated', '1');` — leave this one as-is (it's a flag, not sensitive)

3. In `isLicensed()` (around line 113), update the instance check:
   - `!!decodeLicenseValue(localStorage.getItem('portfolioLicenseInstance') || '')`
   - The decoded value must be truthy (non-empty after decode)

4. In the DOMContentLoaded handler (around line 218), update the stored key read:
   - `storedKey = decodeLicenseValue(localStorage.getItem('portfolioLicenseKey') || '');`

5. **Migration for existing users:** At the top of DOMContentLoaded (before any key reads), add a one-time migration that detects plain-text keys and re-encodes them. Detection: if the value does NOT look like valid Base64 (contains characters outside A-Za-z0-9+/=, or is a UUID-like pattern with hyphens), re-encode it:

```javascript
// One-time migration: encode plain-text keys from pre-v18 installs
(function migratePlainKeys() {
  try {
    ['portfolioLicenseKey', 'portfolioLicenseInstance'].forEach(function(k) {
      var raw = localStorage.getItem(k);
      if (!raw) return;
      try { atob(raw); } catch (e) {
        // Not valid Base64 — encode it
        localStorage.setItem(k, encodeLicenseValue(raw));
      }
    });
  } catch (e) { /* ignore */ }
})();
```

Per D-01, D-02, D-03: This is cosmetic obfuscation only. Base64 is "professional appearance," the real security is LS 2-device activation limit.
