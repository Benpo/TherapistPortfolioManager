4. **Refactor Promise anti-pattern.** Find all instances of `new Promise(async (resolve, reject) => {...})` (around lines 229, 248, 259, 269, 293, 313). Convert each to a plain `async function` pattern:

   Before:
   ```javascript
   function someDBOp(args) {
     return new Promise(async (resolve, reject) => {
       try {
         const db = await openDB();
         // ... use db ...
         resolve(result);
       } catch(e) { reject(e); }
     });
   }
   ```

   After:
   ```javascript
   async function someDBOp(args) {
     const db = await openDB();
     // ... use db ...
     return result;
   }
   ```
