# Phase 05.2 — Landing Page Polish

## Goal
תיקון עיצוב ותוכן בדף הנחיתה לפני לאונץ׳.

---

## Design Changes

### 1. Hero — Aurora background effect
- הרקע של ה-hero כרגע ירוק כהה מדי
- להחליף לאפקט aurora: gradient animation בגוונים ירוק פסטל עדינים
- Vanilla CSS (לא React) — `@keyframes` + `repeating-linear-gradient`
- גוונים: ירוק-פסטל, מנטה, לבן/קרם — לא כחול
- הטקסט נשאר על הרקע, האנימציה ברקע בלבד

### 2. Feature cards — Spotlight glow hover effect
- להוסיף אפקט spotlight ירוק שעוקב אחרי העכבר על כל כרטיסייה
- Vanilla JS: `pointermove` → CSS custom properties `--x`, `--y`
- CSS radial-gradient spotlight בגוון ירוק עדין
- Border glow ירוק כשהעכבר על הכרטיסייה

### 3. Content boxes — Softer glow effect
- לסקשיינים הבאים להוסיף border glow עדין יותר (לא spotlight מלא):
  - "למי מיועד Sessions Garden?"
  - "למה בניתי את זה"
  - Pricing section
- עיגול פינות + הצללה ירוקה + transition רך
- הסקשיין של "למי מיועד" כרגע ירוק כהה ומרובע — לעגל פינות, לעשות מעבר רך

### 4. Section separation — "Why I built this" + Pricing
- כרגע "למה בניתי את זה" ו-pricing נמזגים ללא הפרדה
- להוסיף הפרדה ויזואלית ברורה בין הסקשיינים
- אפשר: background שונה, padding גדול יותר, botanical divider
