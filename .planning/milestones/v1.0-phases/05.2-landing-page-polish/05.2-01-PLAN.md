**Hero aurora background:**

In `assets/landing.css`, define a CSS `@keyframes aurora` animation and add it to the hero section. The current `.landing-hero` has a dark green gradient background — replace it with the aurora effect.

The aurora must use pastel greens, mint, and cream/white tones (NOT blue). Use `repeating-linear-gradient` layered with animation for the shimmer effect. The text must remain legible on top — keep sufficient contrast.

Example approach (adjust values to taste):
```css
@keyframes aurora {
  0%   { background-position: 0% 50%; }
  50%  { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

.landing-hero {
  background: linear-gradient(
    135deg,
    #e8f5e4 0%,
    #d4edda 20%,
    #c8f0e0 40%,
    #e0f5ec 60%,
    #f0faf5 80%,
    #e8f5e4 100%
  );
  background-size: 300% 300%;
  animation: aurora 12s ease infinite;
}
```

Use the existing CSS design tokens where possible (`var(--color-primary)` etc). In dark mode, the aurora should use darker but still soft green tones — add a `[data-theme="dark"] .landing-hero` override.

**Add spotlight glow JavaScript:**

After the `applyLang()` call in the DOMContentLoaded block (or in a separate init function at the bottom of landing.js), add the spotlight listener:

```javascript
// Spotlight glow on feature cards
document.querySelectorAll('.feature-card').forEach(function(card) {
  card.addEventListener('pointermove', function(e) {
    var rect = card.getBoundingClientRect();
    card.style.setProperty('--x', (e.clientX - rect.left) + 'px');
    card.style.setProperty('--y', (e.clientY - rect.top) + 'px');
  });
});
```

Place this after the DOM is ready — wrap in `document.addEventListener('DOMContentLoaded', ...)` or place at the very bottom of the script after `applyLang` is called.
