# Phase 05.2 — Landing Page Polish

## Goal
תיקון עיצוב ותוכן בדף הנחיתה לפני לאונץ׳. נגדיר כאן את כל מה שצריך לשנות.

## What Needs to Change

### עיצוב
1. **האיורים הבוטניים** — הסגנון טוב, הביצוע מכוער. צריך לשפר: בהירים יותר, פחות על הפנים, יותר עדינים.
2. **האייקונים בכרטיסיות** (מסך, מנעול, תיקיה, פנקס, שעון, כוכב) — מכוערים, צריך להחליף לסגנון יפה יותר שמתאים לעיצוב הכולל.

### תוכן / מבנה
3. **סעיף תמונות מסך** — חסר לגמרי. צריך להוסיף סעיף שמציג 3 תמונות מסך מהאפליקציה:
   - Dashboard
   - Session view
   - Client view
   - ספיר תצלם ותשלח את הצילומים (עם לקוחות בדויים)
   - לבנות את הסעיף עם placeholder עד שהצילומים מוכנים
4. **אזור יצירת קשר** — צריך להוסיף contact section לדף
5. **Impressum** — יש placeholder, צריך פרטי קשר אמיתיים (ספיר תספק לפני לאונץ׳)
