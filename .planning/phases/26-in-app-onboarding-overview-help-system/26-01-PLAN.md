| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-26-01 | Information Disclosure | `data-tour` anchor + "?" entry-point design | mitigate | Spec mandates anchors are inert `data-tour` marker attributes carrying NO client/session data, and the tour engine logs nothing; explicitly state the tour never reads or persists PII or session content |

| T-26-04 | Repudiation | n/a (no shared state, no server, no auth, local-only single-user app) | accept | Residual risk LOW — no multi-user or audit surface exists; the help system is read-only guidance over the user's own device data |
