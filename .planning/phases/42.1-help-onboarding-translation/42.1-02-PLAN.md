## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| translated string → tour/popup DOM | \n\n and {ui:}/placeholder tokens are structural; dropping them corrupts rendered layout/labels |
| new content file → service worker cache | A file missing from PRECACHE_URLS breaks help/changelog offline |
| page `<script>` tags → popup locale data | The popup reads window.CHANGELOG_CONTENT_<loc>; a page missing the locale sibling tag renders English regardless of the entries() fix |
| localized version label → hardcoded EN fallback | Dropping {V}/{X.Y} silently routes to an English label with no gate |
