# Phase 42.1 Plan 04: German Help Corpus Summary

Authored `assets/help-content-de.js` — the German (Sie-form) mirror of `help-content-en.js`, a strict structural clone registering `window.HELP_CONTENT_DE` and passing all 10 locale-de assertions in the per-locale integrity gate.

## What was built

A single new file, `assets/help-content-de.js`, containing an IIFE that registers `window.HELP_CONTENT_DE` — a 12-section array structurally byte-identical to `HELP_CONTENT_EN` (same section `id`/`group`/`featured`, topic `id`/`priority`/`covers[]`, body node `type`/glyph `name`, every `{ui:}` token, and all ordering). Only `title` and body `text`/`steps` prose was translated into German using the formal Sie form. The file does NOT redefine `window.HELP_DEEPLINKS` (EN-only), and registers only the DE global (EN content global stays undefined on isolated load).
