Phase 47 — remaining visual details (for sign-off)

Structure & reorder model are already locked (Model A, D-02 order, severity glued-before / free-after). These are the small visual decisions the UI-SPEC pins down: reorder-row look, drag/arrow affordances, the ⓘ severity-off explainer, the N/A rating, and the topic + before-rating row. Each card shows the RECOMMENDED treatment.

1 · Settings reorder list

Recommended: compact rows in one container

Not the big padded settings cards — one bordered list of compact rows so the whole order is scannable. Group headers tinted green; members indented. Drag a handle or use ↑/↓. Try it.

2 · “Severity after” toggle + ⓘ

Recommended: tap-to-open popover (mobile-safe)

This toggle IS the severity-tracking switch (D-08) — not obvious, so a muted ⓘ sits beside it. Tap it (not hover-only) to reveal the explainer. The same text seeds the help entry.

Severity after — track issue severity
Turning this off hides the after-session ratings AND the before-rating column inside each topic — the topics themselves stay. This is how you turn severity tracking off completely.

3 · Severity scale + N/A

Recommended: 11th pill, neutral grey, at the end

N/A joins the 0–10 scale as a wider neutral pill so a therapist who usually rates can skip one session while the field stays mandatory (D-09). Picking N/A hides that topic’s after-rating.

Pick N/A to see its after-rating auto-hide.

4 · Topic + before-rating row

Recommended: rating stacked under the topic

Before-severity is glued to the topic (D-07). Rating sits directly under the topic label. When severity tracking is OFF, the whole rating column vanishes — the topic text remains. Use the switch:
