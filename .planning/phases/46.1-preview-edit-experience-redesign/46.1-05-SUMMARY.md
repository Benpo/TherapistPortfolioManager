# Phase 46.1 Plan 05: Preview/Edit Redesign Copy Contract Summary

Authored the redesigned Preview & Edit copy in all four locales (EN corpus of record + HE/DE/CS parity, RTL-correct Hebrew): added the `toolbar.previewChip` "PREVIEW" key, reconciled the discard-dialog, ephemeral-edit note, empty-state body, and Step-2 helper to the UI-SPEC, switched the steps-1->2 CTA to "Continue", and removed the orphaned `toolbar.hidePreview` key from every locale.

## What was built

- **EN corpus (Task 1).** In `assets/i18n-en.js`: added `toolbar.previewChip` = "PREVIEW"; removed the orphaned `toolbar.hidePreview` (the old target-state toggle owned it); set `export.next1` and `export.next2` to "Continue" (`export.done` left "Done"); reconciled `export.discard.title/body/yes` to "Discard your export edits?" / "These changes apply to this export only and won't be saved." / "Discard" (`export.discard.no` already "Keep editing"); reconciled `export.ephemeralNote` to the UI-SPEC "Restructure freely — nothing is saved back to the session." copy; reconciled `export.step2.helper` to reference switching to Preview; changed `toolbar.previewEmptyBody` to "Start typing above." (`toolbar.previewEmptyTitle` left unchanged = "Nothing to preview yet"). Switcher labels `toolbar.backToEdit` = "Edit" and `toolbar.preview` = "Preview" verified unchanged (reused, no new keys).
- **HTML defaults (Task 1).** In `add-session.html`, the Step-2 `data-i18n="export.step2.helper"` and `data-i18n="export.ephemeralNote"` default paragraph text now match the reconciled EN values.
- **Four-locale parity (Task 2).** Mirrored every added/changed key into `assets/i18n-he.js`, `assets/i18n-de.js`, `assets/i18n-cs.js`: `toolbar.previewChip` added (HE "תצוגה", DE "VORSCHAU", CS "NÁHLED"); `toolbar.hidePreview` removed in all three; `export.next1`/`next2` set to the native "Continue" (HE "המשך", DE "Weiter", CS "Pokračovat") with `export.done` unchanged; discard-dialog, ephemeral-note, Step-2 helper, and empty-state body reconciled to the new EN meaning in each language. Hebrew authored RTL-natural.
