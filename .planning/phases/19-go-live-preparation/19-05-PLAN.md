```javascript
var termsLink = document.getElementById('footer-terms');
```

With:
```javascript

if (impLink) impLink.href = lang === 'de' ? './impressum.html' : './impressum-' + lang + '.html';

```

**Key logic:** German (DE) links go to the primary file (no suffix). All other languages get the `-{lang}` suffix. The `?readonly=true` param on disclaimer links is preserved (it controls the acceptance UI, not the language).
