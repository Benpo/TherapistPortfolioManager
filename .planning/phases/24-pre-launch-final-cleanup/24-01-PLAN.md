| T-24-01-01 | Tampering | `?clientId=` URL param | mitigate | `Number.parseInt(clientId, 10)` validates numeric; falsy / NaN / non-matching id → no-render branch. No SQL or DB query injection vector — IDB lookup is by parsed integer. |
| T-24-01-02 | Information Disclosure | spotlight DOM render (photo, notes, name) | accept | Local-only app, single user, no cross-origin frames render this page. No new disclosure surface introduced. |
