## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| i18n/copy → DOM | Overlay text is set into the DOM; interpolated innerHTML would be an XSS vector. |
| Browser install engine → page | The captured `beforeinstallprompt` event is a browser-mediated object the page defers and later re-fires. |
| localStorage/sessionStorage → coordinator logic | Flags gate which surface shows; a user can clear them (non-adversarial single-user local app). |
