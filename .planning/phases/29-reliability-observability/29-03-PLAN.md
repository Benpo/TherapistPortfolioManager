## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| crash log → clipboard | The redacted log crosses to the system clipboard, where the user pastes it into email |
| crash log → mailto body | A short body crosses to the email client; the full log must not (URL limits + exposure) |
| report assembly → network | Must make zero network calls; transmission is always the user's own mailto action |
| integrity check → crash log | Phase 28's mismatch is persisted locally via CrashLog.logError (no transmission) |
