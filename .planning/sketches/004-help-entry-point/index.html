Discoverability: moderate — sits with two icons users already learned in Phase 25, so the pattern is "all the tools live up here." Intrusion: none — it is chrome, never over content. Consistency: preserves the consolidated single-header shape.

Discoverability: high raw visibility — a coloured circle draws the eye. Intrusion: real — overlaps the last client row and visually competes with content. Consistency: breaks the Phase-25 consolidated-header convention (a second, separate control surface).

The trade-off, plainly. Floating wins on raw "I can see it" visibility, but at the cost of covering content and introducing a second control region the rest of the app does not have. Header-icon trades a little first-glance salience for living exactly where Sessions Garden already taught users to find tools (cloud + gear), with zero content occlusion. For a calm, offline notes app used repeatedly by the same therapist, learned location beats one-time salience.
