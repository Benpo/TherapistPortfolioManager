## Problem

When adding a session, if the user notices the client info needs updating (wrong phone, missing details), they have to leave the session flow, navigate to the client, edit it, then come back to add the session.

## Solution

Add an "Edit Client" button/link on the Add Session screen that opens the client edit form (or navigates to it) so the user can quickly fix client details without losing their session context.
