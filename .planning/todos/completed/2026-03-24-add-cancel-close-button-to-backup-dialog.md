## Problem

The backup dialog (which asks for a password) has no way to dismiss it — there is no X button and no Cancel option. The user is forced to complete the backup with no way to abort or close the dialog.

## Solution

Add a Cancel button (and/or an X close button) to the backup password dialog so the user can dismiss it without completing the backup.
